# Prismic + Next.js 3ap Slice Machine Demo

Working example for Prismic Slice Machine.

4 Slices:

- `CallToAction`
  - variants: `default` and `darkMode`
- `JumboTron`
- `OurService`
- `Testimonial`

## Localhost

- `npm install`
- `npm dev`
- navigate to the `/3ap-slice` path
